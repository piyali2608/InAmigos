## ✨ Features
*   **Fully Responsive UI:** Optimized for seamless viewing across desktop, tablet, and mobile devices.
*   **Dark/Light Theme:** Built-in theme toggle for accessibility and user preference.
*   **Dynamic Visuals:** Automated background hero slider and interactive UI elements.
*   **Performance:** Lightweight frontend built entirely with HTML, CSS, and Vanilla JavaScript.

## 🏛️ The Six Pillars of Change
Our website highlights the core initiatives driving our grassroots mission:
1.  **🍱 Project Seva:** Food and clothing distribution for underprivileged communities.
2.  **📚 Project Bachpanshala:** Bridging the learning gap through tutoring and digital literacy.
3.  **🐾 Project Jeev:** Rescue, medical care, and feeding programs for street animals.
4.  **✊ Project Udaan:** Skill development and financial literacy for women.
5.  **🌳 Project Prakriti:** Tree plantation, water conservation, and climate awareness.
6.  **🛠️ Project Vikas:** Vocational training for youth to build market-ready skills.
